# Agent-Consensus-Case-Data
Multi-agent consensus simulation experimental dataset

## Dataset Overview
This repository stores all raw simulation text data for the multi-agent consensus case study in my paper.

## File Directory Explanation
1. parameter.txt：Global simulation parameter configuration
2. consensus processresult.txt：Full evolution trajectory of consensus iteration
3. evaluation and parameter data.txt：Quantitative evaluation indicators of model performance
4. resource dependence data.txt：Experimental results under resource constraint scenarios
5. comparison-*.txt：Contrast experiment results of different consensus mechanisms
6. comparison-iteration.txt:The consensus change based on different iterations
7. comparison-q.txt:The consensus change based on different values of q
8. comparison-budget.txt:The budget change based on different values of lambda
9. comparison-lambda.txt:The consensus change based on different values of lambda
10. comparison-personlize-demand:The consensus change based on different personlized demands

## Data Usage
All .txt files are raw output of simulation codes, which can be imported into Python / MATLAB to reproduce figures and statistical results in the paper.
